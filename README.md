🕹️ Vanilla Browser Arcade
Welcome to the Browser Arcade! This repository features two highly interactive, fast-paced arcade games built entirely from scratch using pure HTML5 Canvas and Vanilla JavaScript.

✨ Created as part of the Google Student Ambassador Program initiative, entirely built and iterated using Gemini Canvas.

🎮 Play The Games
No downloads, no installations. Click the links below to play instantly in your browser (Works on Desktop and Mobile!).

1. 🍣 Food Slicer: Master Chef Edition
Step into the kitchen! Swipe your blade (cursor/finger) to prep fresh ingredients as they fly across the screen.

Mechanics: Custom line-segment collision detection for lightning-fast swipes, gravity physics, and particle splatter effects.

Warning: Do NOT slice the toxic pufferfish! 🐡

▶️ Play Food Slicer Now

2. ⚔️ Army Clash
A fast-paced, math-based runner strategy game. Steer your leader through mathematical gates to multiply your army and collect weapon crates to upgrade your troops from throwing rocks to firing muskets!

Mechanics: Entity management, automatic projectile firing, scrolling backgrounds, and an epic Boss Clash phase.

🛠️ The Technical Magic: Zero Dependencies
The core philosophy behind these games was pushing the limits of the browser without relying on a single external asset.

🚫 No Image Files: You won't find a single .png or .jpg here. All graphics are rendered using native text emojis, enhanced with Canvas API drop-shadows, brightness filters, and programmatic geometric particles. The custom cursors are generated via inline SVGs.

🚫 No Audio Files: There are no .mp3 or .wav files to download. Every single sound effect—from the sharp shhhk of the chef's knife to the deep synth pulses of a boss fight—is mathematically synthesized in real-time using the browser's native Web Audio API.

⚡ Single-File Architecture: Everything (HTML structure, Tailwind CSS styling, and the complex JavaScript game loop) is bundled into standalone, plug-and-play HTML files.

🚀 Built With
HTML5 Canvas (Rendering & Game Loop)

💡 About the Project
This project was developed as a technical showcase for the Google Student Ambassador Program. The goal was to demonstrate how modern AI tools like Gemini Canvas can accelerate the creative coding process—transforming abstract game mechanics into polished, "juicy" arcade experiences in record time, while keeping the codebase incredibly lightweight.
